# Terraform S3 Bucket

A simple Terraform project that automatically creates an S3 bucket on AWS.

## What it does
* Creates an S3 bucket on AWS using Infrastructure as Code
* No manual clicking in AWS console needed

## Technologies Used
* Terraform
* AWS S3

## How to Run
1. Install Terraform
2. Configure AWS credentials
3. Initialize Terraform:
```bash
terraform init
```
4. Preview changes:
```bash
terraform plan
```
5. Apply:
```bash
terraform apply
```
6. Destroy when done:
```bash
terraform destroy
```
